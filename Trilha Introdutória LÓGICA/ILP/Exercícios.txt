Faça um algoritmo para informar à Joãozinho se ele foi aprovado na matéria de
Lógica para programação. Joãozinho fez 3 provas e a média necessária para ser
aprovado é de 7;


//Entrada de dados:
        Nota1
        Nota2
        Nota3
//Processamento:
        Calcular média

//Saída
        Mostrar se aprovado ou Reprovado

=========================================
Algoritmo:

Leia Nota1
Leia Nota2
Leia Nota3


TotalNotas = Nota1 + Nota 2 + Nota 3
Media = TotalNotas /3

Escreva "Media"

Se Media >= 7 então
        Escreva "aprovado"
senão 
        Escreva "reprovado"
